# Documentation

This folder contains all documentation related to the **Networking Companion** project.  
It includes the following:

- **Requirements** – functional and non-functional requirements of the system  
- **User Stories** – description of user needs and interactions  
- **Tool Justification** – reasons for choosing React + Vite, Python, MongoDB, GitHub, Confluence, Google Drive, Firebase, and Kanban  
- **Coding Convention** – guidelines for coding style and collaboration  
- **Meeting Notes** – summaries of team meetings  
- **Client Notes** – records of client discussions
